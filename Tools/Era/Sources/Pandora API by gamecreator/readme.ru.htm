Pandora's API

версия 2.0.1 (c) gamecreator 2014-2015

Предназначение Pandora's API - сделать добавление пользовательских ERM рисиверов простым и удобным.
SDK можно найти в Mods\Pandora's API\SDK.
Pandora's API реализует следующий функционал:

  • новое событие Эры "OnPandora'sApiReady" для указания когда пользователь может вызывать функции Pandora's API;
    • 

  • пронстранство имён ErmVar в SDK, обеспечивающее доступ ко всем переменным ERM;
    • 

  • функции:

    • int __stdcall RegisterErmReceiver(char name[2], ErmInnerHandler handler, int type)
Добавляет определённый пользователем рисивер к ERM. Параметр type определяет количество параметров рисивера и должен быть равен одному из следующих значений:
0 - без параметров (аналогично UN);
1 - один параметр (аналогично BM);
2 - от одного до пяти параметров (аналогично OB);
3 - один параметр (аналогично FU);
4 - четыре параметра (аналогично DO);
5 - два параметра (аналогично HT).
      • 

    • int __stdcall GenericErmHandler(ErmHandler handler, char c, int num, void* todo, void* mes)
Вспомогательная функция. Переводит все команды в простой формат и вызывает для каждой пользовательский обработчик.
      • 

    • void ConnectPandorasAPI()
Функция SDK. Загружает Pandora's API для пользовательского плагина.
      • 

    • void ApplyArg(ErmArgument& arg, int& value)
Вспомогательная функция SDK. Применяет целочисленный параметр к целочисленной переменной в зависимости от того, как он был передан (установить/проверить/получить значение).
      • 

    • bool CheckArgs(ErmCommand& cmd, ErmArgumentFormat format)
Вспомогательная функция SDK. Проверяет, удовлетворяют ли параметры команды указанным требованиям.
      • 

    • bool IsValidVar(char vartype, int number)
Вспомогательная функция SDK. Проверяет существование переменной ERM с указанным типом и индеском.
    
Мод содержит два рисивера:

    • PA - предназначен для настройки Ящиков Пандоры и Событий (readme)
      • 

    • QU - предназначен для настройки Хижин Провидца и Стражей Прохода (readme)
    
Исходный код этих рисиверов можно найти в Mods\Pandora's API\Sources.
