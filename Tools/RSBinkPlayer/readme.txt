RSBinkPlayer v1.2.5
By Sergey Rozhenko <mailto:sergroj@mail.ru>
https://sites.google.com/site/sergroj/

.bik and .smk player. Can show Heroes 3 videos. You can associate it with .bik and .smk files for more convinience.

Controls:
PaUp, PgDn, Home, End - Previous, next, first, last file in the folder.
Ctrl+Z - Go to previously selected file.
Space - Pause.
Left, Right arrows - Previous, next cadre.
O, L - Load file.
S - Save cadre (if paused) or all cadres (is not paused).
V - Show cadre number in window caption on/off.
H - Halftone stretching on/off.
D - Pause the movie when on deactivate on/off.
